Examen Práctico: Sistema de Login con Favoritos

⏱️ Tiempo: 40 minutos
📝 Materia: Entornos Gráficos

Consigna

Desarrollar un sistema simple de autenticación con gestión de productos favoritos.

⚠️ Importante: Asumir que existe una base de datos con las siguientes tablas:
  • usuarios (id, username, password)
  • productos (id, nombre, precio)

Requisitos del Sistema

1. Página de Login (login.php)

  • Formulario con campos username y password
  • Validar credenciales contra la tabla usuarios mediante consulta SQL
  • Si existe una cookie con el último usuario, pre-completar el campo username
  • Al iniciar sesión correctamente:
    • Guardar el id del usuario en una variable de sesión
    • Crear una cookie que recuerde el username por 7 días
    • Redirigir a la página de productos

2. Página de Productos (productos.php)

  • Verificar que exista sesión activa, sino redirigir a login
  • Realizar consulta SQL para mostrar todos los productos en una tabla HTML
  • Incluir botón/link para cerrar sesión

3. Cerrar Sesión (logout.php)

  • Destruir la sesión activa
  • Redirigir a login

4. Estilos (CSS)

  • Aplicar estilos básicos al formulario de login
  • Estilizar la tabla de productos (colores, bordes, espaciado)
📁 Archivos a entregar:
  • login.php
  • productos.php
  • logout.php
  • styles.css
